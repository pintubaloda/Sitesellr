Source: https://github.com/Kuzma02/Fashion-eCommerce-Shop-in-React

How to import:
1) Zip this folder so root contains:
   - theme.manifest.json
   - assets/
2) Platform Owner -> Platform Themes -> Import Theme ZIP
3) Store Owner -> Store Builder -> Apply "Kuzma Fashion Pro"

Store owner controls (Store Builder -> Catalog Visibility JSON):
{
  "offerBannerText": "Flat 15% off today",
  "offerCtaText": "Grab Offer",

  "offerPercent": 15,
